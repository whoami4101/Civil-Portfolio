# KrishiMitra Landing Page

Smart Farming IoT Device Landing Page

## Features
- Responsive design
- Subscription plans (Bronze, Silver, Gold, Diamond)
- PMDDKY scheme pricing
- App download section
- Modern UI with smooth animations

## Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel
```

## Local Development

Simply open `index.html` in your browser.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons
